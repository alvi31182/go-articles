2. `chan<- T` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç —Ç–∏–ø –∫–∞–Ω–∞–ª–∞ —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä—ã –Ω–µ –ø–æ–∑–≤–æ–ª—è—é—Ç –ø–æ–ª—É—á–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –∏–∑ –∫–∞–Ω–∞–ª–æ–≤ —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏.
3. `<-chan T` –æ–±–æ–∑–Ω–∞—á–∞–µ—Ç —Ç–∏–ø –∫–∞–Ω–∞–ª–∞ —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä—ã –Ω–µ –ø–æ–∑–≤–æ–ª—è—é—Ç –æ—Ç–ø—Ä–∞–≤–ª—è—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –≤ –∫–∞–Ω–∞–ª—ã —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞.

`T` –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è —Ç–∏–ø–æ–º —ç–ª–µ–º–µ–Ω—Ç–∞ —ç—Ç–∏—Ö —Ç–∏–ø–æ–≤ –∫–∞–Ω–∞–ª–æ–≤.

–ó–Ω–∞—á–µ–Ω–∏—è –¥–≤—É–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω–æ–≥–æ —Ç–∏–ø–∞ –∫–∞–Ω–∞–ª–∞ `chan T` –º–æ–≥—É—Ç –±—ã—Ç—å –Ω–µ—è–≤–Ω–æ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω—ã –∫–∞–∫ –≤ —Ç–∏–ø —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ `chan<- T`, —Ç–∞–∫ –∏ –≤ —Ç–∏–ø —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞ `<-chan T`, –Ω–æ –Ω–µ –Ω–∞–æ–±–æ—Ä–æ—Ç (–¥–∞–∂–µ —è–≤–Ω–æ). –ó–Ω–∞—á–µ–Ω–∏—è —Ç–∏–ø–∞ —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏ `chan<- T` –Ω–µ –º–æ–≥—É—Ç –±—ã—Ç—å –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω—ã –≤ —Ç–∏–ø —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞ `<-chan T`, –∏ –Ω–∞–æ–±–æ—Ä–æ—Ç. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –∑–Ω–∞–∫–∏ `<-` –≤ –ª–∏—Ç–µ—Ä–∞–ª–∞—Ö —Ç–∏–ø–æ–≤ –∫–∞–Ω–∞–ª–æ–≤ —è–≤–ª—è—é—Ç—Å—è –º–æ–¥–∏—Ñ–∏–∫–∞—Ç–æ—Ä–∞–º–∏.

–ö–∞–∂–¥–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ –∏–º–µ–µ—Ç –µ–º–∫–æ—Å—Ç—å, —á—Ç–æ –±—É–¥–µ—Ç –æ–±—ä—è—Å–Ω–µ–Ω–æ –≤ —Ä–∞–∑–¥–µ–ª–µ –ø–æ—Å–ª–µ —Å–ª–µ–¥—É—é—â–µ–≥–æ. –ó–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ —Å –Ω—É–ª–µ–≤–æ–π –µ–º–∫–æ—Å—Ç—å—é –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –Ω–µ–±—É—Ñ–µ—Ä–∏–∑–æ–≤–∞–Ω–Ω—ã–º –∫–∞–Ω–∞–ª–æ–º, –∞ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ —Å –Ω–µ–Ω—É–ª–µ–≤–æ–π –µ–º–∫–æ—Å—Ç—å—é –Ω–∞–∑—ã–≤–∞–µ—Ç—Å—è –±—É—Ñ–µ—Ä–∏–∑–æ–≤–∞–Ω–Ω—ã–º –∫–∞–Ω–∞–ª–æ–º.

–ù—É–ª–µ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è —Ç–∏–ø–æ–≤ –∫–∞–Ω–∞–ª–æ–≤ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –ø—Ä–µ–¥–æ–±—ä—è–≤–ª–µ–Ω–Ω—ã–º –∏–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä–æ–º `nil`. –ù–µ–Ω—É–ª–µ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å —Å–æ–∑–¥–∞–Ω–æ —Å –ø–æ–º–æ—â—å—é –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ `make`. –ù–∞–ø—Ä–∏–º–µ—Ä, `make(chan int, 10)` —Å–æ–∑–¥–∞—Å—Ç –∫–∞–Ω–∞–ª, —Ç–∏–ø —ç–ª–µ–º–µ–Ω—Ç–∞ –∫–æ—Ç–æ—Ä–æ–≥–æ ‚Äî `int`. –í—Ç–æ—Ä–æ–π –∞—Ä–≥—É–º–µ–Ω—Ç –≤—ã–∑–æ–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ `make` —É–∫–∞–∑—ã–≤–∞–µ—Ç –µ–º–∫–æ—Å—Ç—å –Ω–æ–≤–æ—Å–æ–∑–¥–∞–Ω–Ω–æ–≥–æ –∫–∞–Ω–∞–ª–∞. –í—Ç–æ—Ä–æ–π –ø–∞—Ä–∞–º–µ—Ç—Ä —è–≤–ª—è–µ—Ç—Å—è –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–º, –∏ –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é ‚Äî –Ω–æ–ª—å.

## –°—Ä–∞–≤–Ω–µ–Ω–∏–µ –∑–Ω–∞—á–µ–Ω–∏–π –∫–∞–Ω–∞–ª–æ–≤

–í—Å–µ —Ç–∏–ø—ã –∫–∞–Ω–∞–ª–æ–≤ —è–≤–ª—è—é—Ç—Å—è —Å—Ä–∞–≤–Ω–∏–≤–∞–µ–º—ã–º–∏ —Ç–∏–ø–∞–º–∏.

–ò–∑ —Å—Ç–∞—Ç—å–∏ [—á–∞—Å—Ç–∏ –∑–Ω–∞—á–µ–Ω–∏–π](https://go101.org/article/value-part.html) –º—ã –∑–Ω–∞–µ–º, —á—Ç–æ –Ω–µ–Ω—É–ª–µ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –∫–∞–Ω–∞–ª–æ–≤ —è–≤–ª—è—é—Ç—Å—è –º–Ω–æ–≥–æ–∫–æ–º–ø–æ–Ω–µ–Ω—Ç–Ω—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏. –ï—Å–ª–∏ –æ–¥–Ω–æ –∑–Ω–∞—á–µ–Ω–∏–µ –∫–∞–Ω–∞–ª–∞ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è –¥—Ä—É–≥–æ–º—É, –¥–≤–∞ –∫–∞–Ω–∞–ª–∞ —Ä–∞–∑–¥–µ–ª—è—é—Ç –æ–¥–Ω—É –∏ —Ç—É –∂–µ –±–∞–∑–æ–≤—É—é —á–∞—Å—Ç—å (—á–∞—Å—Ç–∏). –î—Ä—É–≥–∏–º–∏ —Å–ª–æ–≤–∞–º–∏, —ç—Ç–∏ –¥–≤–∞ –∫–∞–Ω–∞–ª–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –æ–¥–∏–Ω –∏ —Ç–æ—Ç –∂–µ –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π –æ–±—ä–µ–∫—Ç –∫–∞–Ω–∞–ª–∞. –†–µ–∑—É–ª—å—Ç–∞—Ç –∏—Ö —Å—Ä–∞–≤–Ω–µ–Ω–∏—è ‚Äî `true`.

## –û–ø–µ—Ä–∞—Ü–∏–∏ —Å –∫–∞–Ω–∞–ª–∞–º–∏

–ï—Å—Ç—å –ø—è—Ç—å –æ–ø–µ—Ä–∞—Ü–∏–π, —Å–ø–µ—Ü–∏—Ñ–∏—á–Ω—ã—Ö –¥–ª—è –∫–∞–Ω–∞–ª–æ–≤. –ü—Ä–µ–¥–ø–æ–ª–æ–∂–∏–º, –∫–∞–Ω–∞–ª ‚Äî `ch`, –∏—Ö —Å–∏–Ω—Ç–∞–∫—Å–∏—Å –∏ –≤—ã–∑–æ–≤—ã —Ñ—É–Ω–∫—Ü–∏–π —ç—Ç–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω—ã –∑–¥–µ—Å—å.

1. –ó–∞–∫—Ä—ã—Ç—å –∫–∞–Ω–∞–ª, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π –≤—ã–∑–æ–≤ —Ñ—É–Ω–∫—Ü–∏–∏  
```go  
close(ch)  
```  
–≥–¥–µ `close` ‚Äî –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è. –ê—Ä–≥—É–º–µ–Ω—Ç –≤—ã–∑–æ–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ `close` –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ–º –∫–∞–Ω–∞–ª–∞, –∏ –∫–∞–Ω–∞–ª `ch` –Ω–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∫–∞–Ω–∞–ª–æ–º —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞.

2. –û—Ç–ø—Ä–∞–≤–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ, `v`, –≤ –∫–∞–Ω–∞–ª, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å  
```go  
ch <- v  
```  
–≥–¥–µ `v` –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ–º, –∫–æ—Ç–æ—Ä–æ–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –ø—Ä–∏—Å–≤–æ–µ–Ω–æ —Ç–∏–ø—É —ç–ª–µ–º–µ–Ω—Ç–∞ –∫–∞–Ω–∞–ª–∞ `ch`, –∏ –∫–∞–Ω–∞–ª `ch` –Ω–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∫–∞–Ω–∞–ª–æ–º —Ç–æ–ª—å–∫–æ –¥–ª—è –ø—Ä–∏–µ–º–∞. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –∑–¥–µ—Å—å `<-` ‚Äî –æ–ø–µ—Ä–∞—Ç–æ—Ä –æ—Ç–ø—Ä–∞–≤–∫–∏ –≤ –∫–∞–Ω–∞–ª.

3. –ü–æ–ª—É—á–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –∏–∑ –∫–∞–Ω–∞–ª–∞, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å  
```go  
<-ch  
```  
–û–ø–µ—Ä–∞—Ü–∏—è –ø—Ä–∏–µ–º–∞ –∏–∑ –∫–∞–Ω–∞–ª–∞ –≤—Å–µ–≥–¥–∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–æ –∫—Ä–∞–π–Ω–µ–π –º–µ—Ä–µ –æ–¥–∏–Ω —Ä–µ–∑—É–ª—å—Ç–∞—Ç, –∫–æ—Ç–æ—Ä—ã–π —è–≤–ª—è–µ—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ–º —Ç–∏–ø–∞ —ç–ª–µ–º–µ–Ω—Ç–∞ –∫–∞–Ω–∞–ª–∞, –∏ –∫–∞–Ω–∞–ª `ch` –Ω–µ –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∫–∞–Ω–∞–ª–æ–º —Ç–æ–ª—å–∫–æ –¥–ª—è –æ—Ç–ø—Ä–∞–≤–∫–∏. –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –∑–¥–µ—Å—å `<-` ‚Äî –æ–ø–µ—Ä–∞—Ç–æ—Ä –ø—Ä–∏–µ–º–∞ –∏–∑ –∫–∞–Ω–∞–ª–∞. –î–∞, –µ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ —Ç–∞–∫–æ–µ –∂–µ, –∫–∞–∫ —É –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ –æ—Ç–ø—Ä–∞–≤–∫–∏ –≤ –∫–∞–Ω–∞–ª.  
–î–ª—è –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–∞ —Å—Ü–µ–Ω–∞—Ä–∏–µ–≤ –æ–ø–µ—Ä–∞—Ü–∏—è –ø—Ä–∏–µ–º–∞ –∏–∑ –∫–∞–Ω–∞–ª–∞ —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞–µ—Ç—Å—è –∫–∞–∫ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ —Å –æ–¥–Ω–∏–º –∑–Ω–∞—á–µ–Ω–∏–µ–º. –û–¥–Ω–∞–∫–æ, –∫–æ–≥–¥–∞ –æ–ø–µ—Ä–∞—Ü–∏—è —Å –∫–∞–Ω–∞–ª–æ–º –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –∫–∞–∫ –µ–¥–∏–Ω—Å—Ç–≤–µ–Ω–Ω–æ–µ –∏—Å—Ö–æ–¥–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–∏, –æ–Ω–∞ –º–æ–∂–µ—Ç –≤–µ—Ä–Ω—É—Ç—å –≤—Ç–æ—Ä–æ–µ –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω–æ–µ –Ω–µ—Ç–∏–ø–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –±—É–ª–µ–≤–æ –∑–Ω–∞—á–µ–Ω–∏–µ –∏ —Å—Ç–∞—Ç—å –≤—ã—Ä–∞–∂–µ–Ω–∏–µ–º —Å –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏. –ù–µ—Ç–∏–ø–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –±—É–ª–µ–≤–æ –∑–Ω–∞—á–µ–Ω–∏–µ —É–∫–∞–∑—ã–≤–∞–µ—Ç, –±—ã–ª–æ –ª–∏ –ø–µ—Ä–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –æ—Ç–ø—Ä–∞–≤–ª–µ–Ω–æ –¥–æ –∑–∞–∫—Ä—ã—Ç–∏—è –∫–∞–Ω–∞–ª–∞. (–ù–∏–∂–µ –º—ã —É–∑–Ω–∞–µ–º, —á—Ç–æ –∏–∑ –∑–∞–∫—Ä—ã—Ç–æ–≥–æ –∫–∞–Ω–∞–ª–∞ –º–æ–∂–Ω–æ –ø–æ–ª—É—á–∏—Ç—å –Ω–µ–æ–≥—Ä–∞–Ω–∏—á–µ–Ω–Ω–æ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∑–Ω–∞—á–µ–Ω–∏–π.)  
–î–≤–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –ø—Ä–∏–µ–º–∞ –∏–∑ –∫–∞–Ω–∞–ª–∞, –∫–æ—Ç–æ—Ä—ã–µ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è –∫–∞–∫ –∏—Å—Ö–æ–¥–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è—Ö:  
```go  
v = <-ch  
v, sentBeforeClosed = <-ch  
```

4. –ó–∞–ø—Ä–æ—Å–∏—Ç—å –µ–º–∫–æ—Å—Ç—å –±—É—Ñ–µ—Ä–∞ –∑–Ω–∞—á–µ–Ω–∏–π –∫–∞–Ω–∞–ª–∞, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π –≤—ã–∑–æ–≤ —Ñ—É–Ω–∫—Ü–∏–∏  
```go

```  
–≥–¥–µ `cap` ‚Äî –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è, –∫–æ—Ç–æ—Ä–∞—è —É–∂–µ –±—ã–ª–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∞ –≤ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞—Ö –≤ Go. –í–æ–∑–≤—Ä–∞—â–∞–µ–º—ã–π —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤—ã–∑–æ–≤–∞ —Ñ—É–Ω–∫—Ü–∏–∏ `cap` ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ —Ç–∏–ø–∞ `int`.

5. –ó–∞–ø—Ä–æ—Å–∏—Ç—å —Ç–µ–∫—É—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∑–Ω–∞—á–µ–Ω–∏–π –≤ –±—É—Ñ–µ—Ä–µ –∑–Ω–∞—á–µ–Ω–∏–π (–∏–ª–∏ –¥–ª–∏–Ω—É) –∫–∞–Ω–∞–ª–∞, –∏—Å–ø–æ–ª—å–∑—É—è —Å–ª–µ–¥—É—é—â–∏–π –≤—ã–∑–æ–≤ —Ñ—É–Ω–∫—Ü–∏–∏  
